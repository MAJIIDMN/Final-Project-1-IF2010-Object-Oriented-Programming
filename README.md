## Directory

1. `config/`: game configuration files.
2. `data/`: runtime data (if needed).
3. `include/`: header files (`.hpp`).
4. `src/`: source files (`.cpp`).
5. `build/`: build artifacts/object files (generated by `make`).
6. `bin/`: executable output (generated by `make`).
7. `makefile`: build rules.
8. `README.md`: project overview.

## Requirements

- A C++17 compiler (`clang++` or `g++`).
- `make` (GNU Make).

## Instructions

### Installation

Make sure you have a C++17 compiler and GNU Make installed.

**Windows note:** this Makefile expects a Unix-like environment such as **WSL** or **MSYS2/MinGW**.

### Build

```bash
make
```

Output:
- Linux/macOS: `bin/game`
- Windows: `bin/game.exe`

### Run

```bash
make run
```

### Clean

```bash
make clean
```

### Rebuild

```bash
make rebuild
```
